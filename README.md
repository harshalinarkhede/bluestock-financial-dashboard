\# Bluestock Financial Intelligence Platform



\## Project Overview



Bluestock Financial Intelligence Platform is a Django-based web application that provides financial analysis and stock screening capabilities for Indian companies.



The platform allows users to:



\* Browse companies

\* Search companies

\* View company financial details

\* Analyze Profit \& Loss data

\* Compare companies

\* Perform stock screening

\* Access financial APIs

\* View sector-wise analysis



\---



\## Technology Stack



\* Python 3

\* Django

\* PostgreSQL

\* Bootstrap 5

\* Chart.js

\* Django REST Framework

\## Features



\### Dashboard



\* Financial statistics overview

\* Total profit records

\* Company listing

\* Sales Trend Chart

\* EPS Trend Chart



\### Stock Screener

\* Sector-wise company analysis
